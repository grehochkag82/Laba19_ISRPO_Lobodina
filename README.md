## Описание работы

В ходе выполнения лабораторной работы №19 я познакомилась с основами языка программирования JavaScript, изучил его место в web-разработке и провел сравнение базовых концепций JavaScript с языком C#.

### Что было изучено:

1. **Основы синтаксиса JavaScript**
   - Отличия от C# (отсутствие типизации, динамическая природа)
   - Способы запуска JS-кода (в браузере и через Node.js)

2. **Циклы в JavaScript**
   - for, while, do...while
   - Операторы break и continue
   - Вложенные циклы
   - Практическое применение для вывода числовых последовательностей и паттернов

3. **Функции в JavaScript**
   - Классическое объявление функций
   - Функции без return
   - Параметры по умолчанию
   - Функции как значения (аналогия с делегатами в C#)
   - Замыкания (closure)
   - Стрелочные функции (arrow functions)

4. **Массивы в JavaScript**
   - Создание и инициализация массивов
   - Методы работы с массивами (push, pop, shift, unshift)
   - Перебор массивов (for, for...of)
   - Разнотипные массивы (отличие от C#)
   - Поиск элементов (indexOf, includes)

5. **Объекты в JavaScript**
   - Создание объектов без классов (object literal)
   - Доступ к свойствам (через точку и квадратные скобки)
   - Добавление и удаление свойств на лету
   - Методы объектов
   - Перебор свойств (for...in)
   - Вложенные объекты и массивы
